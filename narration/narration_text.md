# Custom narration text

Replace this file with the exact narration you want the Ryan Neural MP3 to read.

When you run the GitHub workflow, set `narration_mode` to `repo_text`. The build will read this file, clean basic Markdown formatting, generate `output/narration_en_gb_ryan.mp3` from this text, and also save the final text it used to `output/narration_text_used.txt`.
